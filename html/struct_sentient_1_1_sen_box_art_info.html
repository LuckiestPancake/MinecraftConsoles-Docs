Detailed Description

Basic box art information.

This structure contains the original uploaded info plus any info implied by the raw data that was uploaded (e.g. data sizes).

SenSysOfferID Sentient::SenBoxArtInfo::GetOfferID ( ) const

Get the offerID associated with this box art.

If command==SenBoxArtCommand_Store, this returns commandData0,1 as a SenSysOfferID. Otherwise, it returns SenSysOfferID_Invalid.
