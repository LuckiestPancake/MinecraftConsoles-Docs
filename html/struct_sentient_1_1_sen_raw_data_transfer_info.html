Detailed Description

Basic raw data transfer information.

This structure contains enough information to retrieve any data. Note: The choice of package vs. file is made internally. Developers may use these struct contents for debugging purposes, but should never rely on anything below the level of this struct and its methods in their own code. All manipulation/exploration should be API-based.

◆ GetBufferSize()

SenSysFileSize Sentient::SenRawDataTransferInfo::GetBufferSize ( ) const

Get the buffer size necessary to download the data.

This will return a size which is greater than or equal to the data size. It may be rounded up somewhat (to, e.g. a 2k boundary) for the purposes of more efficient data transfers. Make absolutely certain that you use THIS size for allocating destination buffers, and NOT GetFileSize().

◆ GetFileSize()

SenSysFileSize Sentient::SenRawDataTransferInfo::GetFileSize ( ) const

Get the actual data size.

Returns the exact size of the data–no rounding. Do NOT use this for allocating destination buffers. Use GetBufferSize() for that.

Test to see if this contains a real file.

Returns true if this structure appears to refer to an actual file, but does not tell you if the file actually exists. Attempt to download the file to determine whether or not it exists.
