{\ttfamily Minecraft.\+Server} is the dedicated-\/server executable entry for this codebase.

Core responsibilities:\+
\begin{DoxyItemize}
\item Switch the process working directory to the executable folder before relative file I/\+O
\item Load, normalize, and repair {\ttfamily server.\+properties}
\item Initialize dedicated runtime systems, connection logging, and access control
\item Load or create the target world and keep {\ttfamily level-\/id} aligned with the actual save destination
\item Run the dedicated main loop (network tick, XUI actions, autosave, CLI input)
\item Maintain operator-\/facing access files such as {\ttfamily banned-\/players.\+json} and {\ttfamily banned-\/ips.\+json}
\item Perform an initial save for newly created worlds and then shut down safely
\end{DoxyItemize}\hypertarget{md__c_1_2_users_2_administrator_2_documents_2_git_hub_2_minecraft_consoles_2_minecraft_8_server_21e77ccb9dc81339e9134109d9b7c205_autotoc_md12}{}\doxysection{\texorpdfstring{2. Main flow in {\ttfamily \doxylink{_server_main_8cpp}{Windows64/\+\+Server\+Main.\+cpp}}:\+
\begin{DoxyEnumerate}
\item {\ttfamily Set\+Exe\+Working\+Directory()} switches the current directory to the executable folder.
\item Load and normalize {\ttfamily server.\+properties} via {\ttfamily \doxylink{_server_main_8cpp_ac8490b313a587bb21427e8f4e306f5da}{Load\+Server\+Properties\+Config()}}.
\item Copy config into {\ttfamily \doxylink{struct_dedicated_server_config}{Dedicated\+Server\+Config}}, then apply CLI overrides ({\ttfamily -\/port}, {\ttfamily -\/ip}/\+{\ttfamily -\/\doxylink{_durango_2_iggy_2gdraw_2gdraw__d3d1x__shared_8inl_ab62f000ffddc47ca0bd8d857c1c7b44b}{bind}}, {\ttfamily -\/name}, {\ttfamily -\/maxplayers}, {\ttfamily -\/seed}, {\ttfamily -\/loglevel}, {\ttfamily -\/help}/\+{\ttfamily -\/-\/help}/\+{\ttfamily -\/h}).
\item Initialize process state, {\ttfamily Server\+Log\+Manager}, and {\ttfamily Access::\+\+Initialize("{}."{})}.
\item Initialize window/\+device/\+profile/\+network/\+thread-\/local systems.
\item Set host/\+game options from {\ttfamily \doxylink{struct_server_properties_config}{Server\+Properties\+Config}}.
\item Bootstrap world with {\ttfamily Bootstrap\+World\+For\+Server(...)}.
\item If world bootstrap resolves a different normalized save ID, persist it with {\ttfamily \doxylink{_server_main_8cpp_a85c299aee8bf056de8ac22a3116a0825}{Save\+Server\+Properties\+Config()}}.
\item Start hosted game thread ({\ttfamily Run\+Network\+Game\+Thread\+Proc}).
\item If a brand-\/new world was created, explicitly request one initial save.
\item Enter the main loop:\+
\begin{DoxyItemize}
\item {\ttfamily Tick\+Core\+Systems()}
\item {\ttfamily Handle\+Xui\+Actions()}
\item {\ttfamily server\+Cli.\+Poll()}
\item autosave scheduling
\end{DoxyItemize}
\item On shutdown:\+
\begin{DoxyItemize}
\item stop CLI input
\item request save-\/on-\/exit /\+ halt server
\item wait for network shutdown completion
\item terminate log, access, network, and device systems
\end{DoxyItemize}
