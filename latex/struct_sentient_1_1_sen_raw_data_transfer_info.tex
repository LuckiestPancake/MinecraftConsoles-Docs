\doxysubsection{Detailed Description}
Basic raw data transfer information. 

This structure contains enough information to retrieve any data. Note:\+ \doxylink{requirements_The}{The} choice of package vs. file is made internally. Developers may use these struct contents for debugging purposes, but should never rely on anything below the level of this struct and its methods in their own code. All manipulation/\+exploration should be API-\/based.
