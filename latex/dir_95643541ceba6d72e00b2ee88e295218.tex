LAN multiplayer is available on the Windows build


\begin{DoxyItemize}
\item Hosting a multiplayer world automatically advertises it on the local network
\item Other players on the same LAN can discover the session from the in-\/game Join Game menu
\item Game connections use TCP port {\ttfamily 25565} by default
\item LAN discovery uses UDP port {\ttfamily 25566}
\item Add servers to your server list with the in-\/game Add Server button (temp)
\item Rename yourself without losing data by keeping your {\ttfamily uid.\+dat}
\item Split-\/screen players can join in, even in Multiplayer
