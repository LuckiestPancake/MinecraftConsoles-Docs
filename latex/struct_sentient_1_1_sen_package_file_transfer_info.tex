\doxysubsection{Detailed Description}
Basic file transfer information. 

Resource-\/specific data structures use this, or a \doxylink{struct_sentient_1_1_sen_file_transfer_info}{Sen\+File\+Transfer\+Info}, to indicate how one can transfer a file to the console. \doxylink{requirements_The}{The} internal package file download routine takes this structure directly.
