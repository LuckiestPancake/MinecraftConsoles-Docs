\item Returns {\ttfamily Failed} when startup should be aborted サーバー起動時のワールド確定処理
\end{DoxyItemize}

{\bfseries{Bootstrap Target World For Server Startup}}

Resolves whether the target world should be loaded from an existing save or created as new
\begin{DoxyItemize}
\item Applies {\ttfamily level-\/name} and {\ttfamily level-\/id} from {\ttfamily server.\+properties}
\item Loads when a matching save exists
\item Creates a new world context only when no save matches サーバー起動時のロードか新規作成かを確定する
\end{DoxyItemize}
