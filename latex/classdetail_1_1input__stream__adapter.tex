\doxysubsection{Detailed Description}
\doxylink{class_input}{Input} adapter for a (caching) istream. Ignores a UFT \doxylink{class_byte}{Byte} Order Mark at beginning of input. Does not support changing the underlying std::\+streambuf in mid-\/input. Maintains underlying std::\+istream and std::\+streambuf to support subsequent use of standard std::\+istream operations to process any input characters following those used in parsing the JSON input. Clears the std::\+istream flags; any input errors (e.\+g., EOF) will be detected by the first subsequent call for input from the std::\+istream.
