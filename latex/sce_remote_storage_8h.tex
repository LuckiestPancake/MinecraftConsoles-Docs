int32\+\_\+t sce\+Remote\+Storage\+Term (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Terminates the Remote\+Storage library. 

Terminates the Remote\+Storage library and deletes the thread that process requests. This method must be executed to terminate the Remote\+Storage library to prevent leaks in memory and resources. This method will abort any other pending requests and terminate the library. It won\textquotesingle{}t wait for requests to finish. This method is synchronous and does not make use of the callback to inform the user of success termination. It is executed on the calling thread.
