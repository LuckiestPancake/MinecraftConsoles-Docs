# ArrowFireEnchantment

## Purpose
Handles the "Flame" enchantment for bows, which causes arrows to be on fire and ignite targets.

## Key Classes
- **ArrowFireEnchantment**: Inherits from `Enchantment`.

## Modding Notes
Sets the arrow entity on fire when shot.

## Important Dependencies
- `Enchantment.h`
