# `.wsuo` (Binary File)

## Purpose
The `.wsuo` file, typically found in Visual Studio's `.vs` directory, stores workspace user options. These options can include things like user-specific settings for a workspace, such as editor configurations, debugger settings, or other preferences that are not part of the solution or project files themselves and are not meant to be shared.

## Key Classes/Functions
N/A: This is a binary configuration file and does not contain code.

## Modifiability for Modders
As a binary file, this should not be modified directly by users. It is managed by Visual Studio and contains proprietary data. Modifying it could corrupt the workspace state.
