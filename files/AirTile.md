# AirTile

## Purpose
Represents the "Air" block in the game world. It is a non-solid, invisible block that occupies empty space.

## Key Classes
- **AirTile**: Inherits from `Tile`.

## Important Functions
- `AirTile()`: Constructor that initializes the tile with `Material::air`.

## Modding Notes
Air is the default state for "empty" coordinates in the world.

## Important Dependencies
- `Tile.h`
- `Material.h`
