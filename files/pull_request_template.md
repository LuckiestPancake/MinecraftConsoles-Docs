# `pull_request_template.md`

## Purpose
This file provides a template for creating pull requests (PRs) in the repository. It guides contributors on how to describe their changes, explain the root cause of issues, detail the fix implementation, and disclose any AI usage. It also emphasizes the importance of attaching gameplay footage for changes affecting game behavior and warns against submitting AI-generated code.

## Key Classes/Functions
This file does not contain code and therefore has no key classes or functions. It is a markdown file used for structuring PR information.

## Modifiability for Modders
As a markdown file, it can be modified by anyone with write access to the repository to better suit the project's needs for PR submission guidelines.
