# lce_filesystem

## Purpose
The `lce_filesystem` folder contains basic utility functions for filesystem interaction, such as checking for the existence of files and folders and iterating through directory contents.

## Contents
- [lce_filesystem.h](../files/lce_filesystem.h.md): Header file for filesystem utilities.
- [lce_filesystem.cpp](../files/lce_filesystem.cpp.md): Implementation of filesystem utilities.

## Modding Impact
Used across the engine for asset loading and path verification.
